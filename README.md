📊 Fundamental Booster – Personal Data Collector
📌 Overview
Fundamental Booster is a simple Python-based interactive program that collects basic user information such as name, age, height, and favorite number. It also demonstrates Python fundamentals like data types, memory addresses, and basic calculations.
🚀 Features
🧑 Collects user details (name, age, height, favorite number)
🔍 Displays:
Data type of each input
Memory address of stored values
🧮 Calculates birth year (based on input age)
📚 Great for beginners learning Python basics
🛠️ Technologies Used
Python 3
